## Task 2 — Calculator

A functional calculator built with vanilla HTML, CSS and JavaScript.

### Features
- Basic arithmetic - add, subtract, multiply, divide
- Live expression display (shows what you're calculating as you type)
- Delete key and full clear
- Handles edge cases - divide by zero returns an error, multiple decimals blocked, chained operations calculate before applying the next operator

### Files
CodeAlpha_Calculator/
├── index.html
├── styles.css
└── script.js

### How to run
Download the three files into one folder and open `index.html` in your browser. That's it.
