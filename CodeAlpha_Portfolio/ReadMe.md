## Task 3 — Personal Portfolio Site

A personal portfolio built to showcase my work, skills and background.

### Live Site
🔗 [dantunmibi.netlify.app](https://dantunmibi.netlify.app)

### Sections
- **Hero** - intro and quick links
- **About** - background, what I do, quick stats
- **Skills** - languages, frameworks, tools
- **Projects** - Rubies Unleashed, PDFnBooks, Atmos Weathr
- **Resume** - experience, education, CV download
- **Contact** - email, X, GitHub, LinkedIn

### Features
- Fully responsive - mobile menu included
- Scroll reveal animations on every section
- Sticky nav that picks up a background on scroll
- Hover effects throughout - cards lift, tags highlight, contact rows slide
- CV download links directly to the resume file
- Smooth scroll via CSS
- Deployed on Netlify

HTML · CSS · JavaScript
